# @amlplugins/elevenlabs-music

Native `@elevenlabs/elevenlabs-js` SDK re-exported under the `@amlplugins` namespace.

> ElevenLabs Music — text-to-music generation with vocals, instrumentation, and prompt-driven composition.

## Install

```bash
npm install @amlplugins/elevenlabs-music
```

## Use

```ts
import { /* named exports from @elevenlabs/elevenlabs-js */ } from "@amlplugins/elevenlabs-music";
// or
import sdk from "@amlplugins/elevenlabs-music";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
